# Data structure based arcade games:
This a project that contains 2 data structure based games binary tree and a stack, it was built in 1 month for an internship, to help learners grasp stack and binary tree concepts in as easier fun way.

- Used software:
  - Unity Game Engine.
 
# Features:
- Stack Game:
   - When interacting at the table by pressing 'E' a main menu appears which displayes 3 options for words (1, 2, 3) with 3 Data structures available: Stack, Queue, Array.
